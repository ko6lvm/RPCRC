# RPCRC/firmware/src
This directory includes the code written by KO6LVM that is built from CircuitPython and KMK.

# RPCRC/firmware/upload
This directory includes all files that should be uploaded into the RP2040 board after flashing the UF2.

# RPCRC/firmware/kmk
This directory is the source for the KMK CircuitPython package. It includes the latest release, commit 8afe569.

The package included is directly from the KMK Github. The latest version can be downloaded at https://github.com/KMKfw/kmk_firmware.

All firmware copyright KMKfw 2026.

# RPCRC/firmware/cpy
This directory includs CircuitPython v10.2.0-alpha.1 for the RP2040/Pi Pico.

The firmware included is directly from CircuitPython. The latest version can be downloaded at https://circuitpython.org/board/raspberry_pi_pico/. To install, drag-and-drop the uf2 file into your RP2040 Mass Storage device in BOOTSEL mode.

All firmware copyright Adafruit Industries 2026.
